# 背景画像フォルダ

このフォルダの画像が**季節 × 時間帯**の組み合わせで自動切替されます。
画像は薄く(白オーバーレイ約90%)背景として表示されます。

## ファイル名規則

**`<季節>　<時間帯>.png`**(間は**全角スペース**)

### 季節(月)

| 季節 | 月 |
|---|---|
| 春 | 3, 4, 5 |
| 夏 | 6, 7, 8 |
| 秋 | 9, 10, 11 |
| 冬 | 12, 1, 2 |

### 時間帯

| 時間帯 | 適用時刻 |
|---|---|
| 朝 | 5:00 〜 9:00 |
| 昼 | 9:01 〜 18:00 |
| 夜 | 18:01 〜 翌 4:59 |

### 必要なファイル(全12枚)

```
春　朝.png  春　昼.png  春　夜.png
夏　朝.png  夏　昼.png  夏　夜.png
秋　朝.png  秋　昼.png  秋　夜.png
冬　朝.png  冬　昼.png  冬　夜.png
```

## 画像仕様

- **形式**: PNG / JPG / WebP
- **解像度**: 1920×1080 以上(横長推奨)
- **ファイルサイズ**: 500KB以下が望ましい(大きいとロード遅延)
- **色味**: 背景として使うので**派手すぎないもの**が読みやすい

## 未設置時の挙動

対応する季節×時間帯の画像がない場合、デフォルトのグラデーションのみが表示されます(エラーにならない)。サイドバーの「Background」欄に `画像未設置: 〇〇.png を配置` と表示されます。
